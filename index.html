Installation

Telechargez et installez les logiciels suivants:

Les logiciels ici sont les minimum requis

Le SDK de ActiveHome Pro

Permet d'obtenir l'ActiveX activehome qui controlera votre CM15Pro

Visual C++ 2008 Express

Permet de compiler la librairie win32ole

Python 2.7.9

Permet de compiler la librairie win32ole


Installez le module node win32ole

Lancer une console (Touche Windows + R puis cmd)

Naviguez vers votre dossier contenant SARAH

Ecrivez NodeJS\npm install win32ole

Il se peut que l'installation renvoie des erreurs

Verifiez que node-gyp est dons vos modules

Si une erreur vous indique qu'un fichier "node.lib" ne peut pas etre lu:

Déplacez node.lib de C:\Users\xxx\.node-gyp\0.10.12\ia32 vers C:\Users\xxx\.node-gyp\0.10.12\

Une fois compilée, déplacez la librairie vers le dossier lib du module

Configuration

Editer ligth.xml pour ajouter vos modules

Dans la premiere balise one-of ajoutez vos commandes

type: sendplc ou sendrf; valeur: on ou off (ou xdim --> a tester)

Dans la deuxieme balise one-of faite correspondre les modules aux emplacements que vous énoncerez
